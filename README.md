# BMAD Test Repository 003

## Overview
This is the BMAD (Business Model Architecture Design) SDLC application repository - a comprehensive platform for managing software development lifecycle processes.

## Description
SDLC App Repository - A full-stack application designed to streamline and automate software development lifecycle workflows, from ideation to deployment.

## Features
- Project management and tracking
- SDLC phase management (Ideation, Planning, Development, etc.)
- Integration with GitHub for version control
- User authentication and authorization
- Document generation and management
- Team collaboration tools

## Technology Stack
- **Frontend**: TypeScript/React
- **Backend**: Node.js, Fastify
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: MSAL (Microsoft Authentication Library)
- **Containerization**: Docker

## Project Structure
```
├── src/
│   ├── backend/     # Backend API services
│   ├── frontend/    # Frontend application
│   └── shared/      # Shared types and utilities
├── docs/            # Documentation
├── scripts/         # Database and setup scripts
└── SDLC-DEMO-HTML/  # Demo HTML pages
```
